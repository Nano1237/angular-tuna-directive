How to use?

For a simple "time since" use the directive like this

<p tuna="12/02/2014">since {{years}} Years</p>

If you want a more accurate output, use it like this

<p tuna="12/02/2014 23:45">since {{years}} Years and {{minutes}} Minutes</p>

How does it work?

Pretty simple, it just passes the Input Parameter of the tuna attribute to the new Date() Constructor.

This Constructor returns a Date Object which returns a Javascript timestamp with the getTime() Method.

Using the timedifference between now and this timestamp, we can then calculate the times past.
